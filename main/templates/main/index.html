How it works

  1. Click the button below
    This takes you to Open Humans.
  2. Log in or create an Open Humans account
    Upon completion, your googlefit data will be added to this account.
  3. Authorize this app in Open Humans
    This authorizes us to add data to your account.
  4. Return to this app
    Come back to the app so we can get your personalized googlefit data.
  5. Click the "connect GoogleFit" button below
    This will redirect you to GoogleFit to authorize so we can fetch GoogleFit data to add to your account.
  6. Authorize with GoogleFit
  7. Return and the app will add data.
    Demo data should be added to your Open Humans account. You'll be able to access & manage data here: {{ oh_proj_page }}
